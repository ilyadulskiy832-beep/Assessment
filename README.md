# DecryptCode INC — Blockchain Full-Stack Developer Assessment

This repository contains a technical assessment for the **Part-Time Blockchain Full-Stack Developer** role. Complete all tasks in **2–3 hours** and submit your solution as instructed below.

## Overview

You will complete **3 tasks** that reflect real work at DecryptCode:

| Task | Focus | Est. time |
|------|--------|-----------|
| **Task 1** | Smart contract (Solidity) | ~45–60 min |
| **Task 2** | Backend API (Node.js/Express) + blockchain | ~45–60 min |
| **Task 3** | Frontend (dApp UI with ethers + optional API) | ~45–60 min |

## Prerequisites

- Node.js 18+
- npm or yarn
- Basic familiarity with Solidity, Ethereum (or EVM), and Node.js/Express
- (Optional) Hardhat or Foundry for local contract development

## Repository structure

```
decryptcode/
├── README.md                 ← You are here
├── task-1-smart-contract/    ← Solidity task
├── task-2-backend-api/       ← Node.js/Express API task
└── task-3-integration/       ← Frontend (dApp UI) task
```

## How to complete the assessment

1. **Clone or download** this repo and work in your own branch or copy.
2. **Complete each task** in its folder according to the task-specific README.
3. **Do not** use external code generators (e.g., full app generators) for the core logic; your own code is what we evaluate.
4. **Document** any setup steps or assumptions in each task folder (e.g., a short `NOTES.md` or in the task README).

## Submission

- Push your solution to a **private** GitHub/GitLab repository.
- Add the DecryptCode contact (provided in your application) as a collaborator, or share a **zip** of the repo (including `node_modules` only if required for running; otherwise exclude it and include clear install/run instructions).
- In the repo root, add a **SUBMISSION.md** with:
  - Your name and email
  - Total time spent (per task optional)
  - How to run each task (commands + any env vars)
  - Any assumptions or shortcuts you took

## Questions

If anything in the instructions is unclear, document your interpretation and proceed. We value clear communication and practical decisions.

Good luck.
